# Amazon Sales Dashboard

## Overview
This is a Power BI dashboard built on Amazon sales data.  
It shows key metrics like sales, profit, orders, and customer segments.

## Features
- Sales, profit, and order KPIs
- Customer segmentation (Consumer, Corporate, Home Office)
- Orders by payment and shipping mode
- Sales by product category
- Monthly sales trends
- Top profitable sub-categories
- Return status and geographic distribution

## Skills Shown
- Power BI dashboard design
- Data modeling and visualization
- KPI tracking and reporting
